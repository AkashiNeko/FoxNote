::: info Linux

[<HopeIcon icon="/icon/linux/linux.svg" size="1rem" /> Linux](https://zh.wikipedia.org/wiki/Linux)是一种开源的、基于UNIX的操作系统，以其自由度、稳定性和安全性而著名。它被广泛应用于服务器、嵌入式设备和个人计算机等领域，为用户提供强大的自定义能力和广泛的软件支持。Linux的开源特性使得用户能够自由地查看、修改和定制系统源代码，满足各种需求，并且拥有一个活跃的开发者社区，提供广泛的软件支持和技术交流。其稳定性和可靠性使得Linux成为托管关键任务和高负载环境的首选，它能够长时间运行而不需要重启，并且能够处理大量的并发任务。

:::
