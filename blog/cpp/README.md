::: info C++

[<HopeIcon icon="/icon/cpp/cpp.svg" size="1rem" /> C++](https://zh.wikipedia.org/zh-hans/C%2B%2B)是一种通用的高级编程语言，它是由 Bjarne Stroustrup 于 1979 年在贝尔实验室开始设计开发的。它扩展自 C 语言，并且具有面向对象编程和泛型编程的特性。C++编译器可在多种操作系统上运行，包括 Windows、Linux、macOS 等。C++ 被广泛应用于系统软件、游戏开发、嵌入式系统、科学计算等各种领域。

:::
